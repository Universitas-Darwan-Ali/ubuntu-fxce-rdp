# Cloud PC
## ubuntu-fxce-rdp
Docker Ubuntu 20.04 with fxce and RDP on web browser

## Installation
[1] Login into Google Cloud Shell
 <https://console.cloud.google.com>
 <br>
[2] Copy this code on Cloud Shell or your Linux Terminal
> git clone https://github.com/Universitas-Darwan-Ali/ubuntu-fxce-rdp && cd ubuntu-fxce-rdp && docker-compose up -d;
